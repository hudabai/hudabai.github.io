其实，我也喜欢你啊

发表于 2015-10-13   |   分类于 想太多   |     |   Hits

昨晚，L小姐给我留了一个悬念，明天给你讲一个关于缘分的故事。

昆明的天空还飘着小雨，在阴冷的马概教室，无聊的翻着手里的莎士比亚十四行诗，我承认伟大的莎士比亚战胜了毛爷爷，然后我突然想起昨晚那个搁置一旁的故事，于是我便缠着L小姐问，故事的开头来源于前几天在刷空间的时候，看到表白墙上有L小姐的匿名表白，于是顺手截图发给她，然后坏笑的看着她问她什么情况，其实我的主要目的是要糖吃。当然不出所料和大多数匿名告白的同志一样，仅仅是暗地里表白一下心意，正主还不知道是谁。

然后，本着宁死不屈，除非给糖的精神，我在表白墙上发了几句”刺激”某先生的话，出乎意料的是，某先生真的向L小姐告白了!告白了！

千千万万个你以为，故事偏偏给你一个没想到。某先生竟然是L小姐大一时暗恋的对象，就在前几天L小姐还在玩大冒险时输掉请某先生去看电影，同时某先生也请L小姐帮选好朋友礼物为由荡了一天。（我相信买礼物不是重点）

故事的结局很让人暖心，两个互相暗恋的人最终在一起了，但是生活中，更多的情况是，当你发现你们彼此暗恋时，时间已经偷走你所有的选择了。曾经你喜欢的人终于对你说了一句，那时候我也喜欢你啊。于是你整个人就懵住了，可是也只能说一句：“恩。”没后悔没遗憾也没有无奈。

曾经有的心动错过时机，也就没了在一起的机会。

我们都明白这个道理。

想说的话当时不说，也就没了再说的必要。

我们都在等待最好的时机，去做那些对的事，结果反而耽误了所谓的时机。

你现在做的事，都是正确的事，包括喜欢一个对的人。

只要有那个人，什么样的时机都是对的。

祝福每个能看到这篇故事的你。：）
